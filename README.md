# ft_ping

## Info

This project is about recoding the ping command.

- Status: finished
- Result: 125%
- Observations: taken `inetutils-2.0` implementation as reference.

## Compile and use

```bash
make
```

```bash
./ping [OPTIONS...] HOST
```

## Features

Run `./ft_ping -h` to see a full list of all options.
